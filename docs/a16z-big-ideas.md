## How to use this file

1. After founder intake, scan keywords against startup description.
2. If match found: surface it to founder. Confirm they want to lean into it.
3. If confirmed: weave the alignment explicitly into startup description and pitch deck framing.
4. Direct partner RFS = highest signal. Name the theme in the answer if alignment is real.

**Do not fabricate alignment.** If the fit is a stretch, say so. Weak alignment is worse than none.

## Alignment scoring guide

Use when checking a founder's startup against these themes:

- **Direct match** — startup is exactly what the partner described. Name the partner and theme in your framing recommendation.
- **Strong overlap** — startup addresses 3+ keywords from a theme. Recommend leaning into the framing.
- **Weak overlap** — 1-2 keyword matches. Do not force-fit. Note it, let founder decide.
- **No match** — proceed normally. Do not manufacture alignment.
